30/12/2024, 20:00 - Francois Bocquet: Petit bémol sur la notion de « données sensibles » qui est mal utilisée dans cette réponse : ce terme est utilisé dans le RGPD pour désigner les données qu’il est interdit de traiter par défaut sauf si nécessité spécifique et statut explicitement autorisé. Les données sensibles sont des données à caractère personnel qui font apparaître, directement ou indirectement :
les origines raciales ou ethniques,
les opinions politiques, philosophiques ou religieuses
l'appartenance syndicale des personnes,
Les données relatives à la santé des personnes,
Les données relatives à la vie sexuelle des personnes,
Les données relatives aux infractions pénales, aux condamnations
Les données biométriques
Les données génétiques
Les données de santé nécessitent en outre un consentement.
