18/11/2024, 21:00 - Francois Bocquet: Du nouveau chez Mistral ce soir -> Voici un résumé des principales nouveautés du chat de Mistral :
Gestion des Dates : Le chat est très attentif aux dates. Par exemple, si vous mentionnez "hier", il comprendra que vous parlez du dimanche 17 novembre 2024. Il s'assure également de fournir des informations pertinentes pour la date spécifiée.
Clarté et Précision : Si votre question n'est pas claire ou manque de contexte, le chat vous demandera des clarifications avant de répondre. Par exemple, si vous demandez "Quels sont les bons restaurants près de chez moi ?", il vous demandera "Où êtes-vous ?".
Multilinguisme : Le chat peut répondre dans la langue que vous utilisez ou que vous demandez. Il suit les instructions dans toutes les langues.
Capacités Multimodales : Le chat peut lire des images mais ne peut pas générer d'images. Il ne peut pas non plus transcrire des fichiers audio ou des vidéos.
Absence de Recherche Web : Le chat ne peut pas effectuer de recherches sur le web ou accéder à Internet pour ouvrir des URL ou des liens. Si vous avez besoin d'informations spécifiques, il vous demandera de copier-coller le texte directement dans le chat.
Pas d'Accès au Mode Canvas : Le chat n'a pas accès au mode de génération de canvas.
Ces fonctionnalités visent à améliorer l'interaction et la précision des réponses fournies par le chat de Mistral.
