Test d'intégration ActiBot

Cette page démontre comment intégrer ActiBot sur n'importe quel site web.

Comment ça marche ?

ActiBot peut être facilement intégré à n'importe quel site web en ajoutant un simple script JavaScript. L'assistant apparaît sous forme d'une icône de chat flottante dans le coin inférieur droit de la page.

L'intégration est légère, responsive et ne nécessite aucune dépendance externe.

Comment l'intégrer dans votre site

Ajoutez simplement ce script à votre site :

<script src="https://votre-domain.com/actibot-embed.js"></script>

Vous pouvez également configurer l'URL de l'API :

<script>
  window.ACTIBOT_API_URL = 'https://actibot.up.railway.app'; // Optionnel, URL par défaut si non spécifiée
</script>
<script src="https://votre-domain.com/actibot-embed.js"></script>

Testez maintenant !

Cliquez sur l'icône de chat dans le coin inférieur droit pour interagir avec ActiBot. Posez-lui une question sur l'intelligence artificielle, la génération d'images, ou tout autre sujet lié au domaine !
